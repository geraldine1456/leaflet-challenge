# 📍Leaflet Earthquake Visualization Challenge 
Visualizing real-time seismic activity using Leaflet.js and USGS data

## 🌍 Overview  
This challenge leverages **Leaflet.js**, **D3.js**, and **GeoJSON** to build an interactive earthquake map using real-time data from the **United States Geological Survey (USGS)**. The visualization helps educate the public, support scientific research, and illustrate global seismic activity trends through dynamic mapping.

---

## ⚙️ Technologies Used

* **Leaflet.js** – Interactive maps and layers
* **D3.js** – Fetching and parsing GeoJSON data
* **GeoJSON** – Earthquake and tectonic plate data format
* **HTML/CSS/JavaScript** – Core web development
* **OpenStreetMap & Esri** – Tile layers for map views

## 🚀 Installation & Setup  
### **1. Clone the Repository**
```bash
git clone https://github.com/geraldine1456/leaflet-challenge.git
```

### **2. Navigate to the Challenge Directory**
```bash
cd leaflet-challenge
```

### **3. Launch the Visualization**

Open the appropriate HTML file in your browser:

* `Leaflet-Part-1/index.html`
* `Leaflet-Part-2/index.html`
---

## 🖥️ Usage

### **Basic Earthquake Map (`Leaflet-Part-1`)**

* View earthquake markers sized by magnitude.
* Click markers to view details: **magnitude**, **depth**, and **location**.
* Use the **color-coded legend** to interpret depth.

###  **Enhanced Map Features (`Leaflet-Part-2`)**

* Toggle between base maps: **Satellite**, **Grayscale**, and **Outdoors**.
* Enable or disable **tectonic plate boundaries**
* Combine plate boundaries with earthquake data for deeper seismic insight.
